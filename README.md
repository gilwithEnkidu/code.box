# 프로젝트 제목
raw data 생성기

## 프로젝트 소개
가상의 IC chromatography 에서 측정된 raw data를 csv 파일로 export 했다는 가정 하에
그 raw data를 생성해주는 프로그램.
이 생성기는 시뮬레이션 데이터 생성, 데이터 추출 및 리포트 생성까지의 파이프 라인 구축의 첫 단계

## 실행 방법
파이썬으로 코드를 실행.
실행 전 코드의 20번째 줄부터 시작되는 사용자 설정에서 생성 일자 및 이상치 발생 확률 등을 설정하면 됨.

## 기능 - 사용자 설정
- 기간설정 : 시뮬레이션하고 싶은 기간을 설정
시작일, 종료일, 일일 최대 생성 개수 등을 설정

- 이상치 : 발생 확률
이상치 발생 확률, 이상치 편차 크기 등을 설정.
각 피분석원의 측정값들은 특정 기대값과 표준편차를 갖는 정규분포를 따른다고 가정하고
코드 실행 시 생성되는 데이터들 중 0.5~0.9% 가 이상치(outlier)인 데이터로 시뮬레이션 함.
따라서 불량 발생 확률을 상정하여 그에 맞는 수치를 입력

## 개발 환경
- 'python 3.7.6'
- 패키지 추가 'numpy'
